# ALGORITHM VISUALIZER

## 🌐 Website Preview

![Preview](preview.gif)

## 🧠 Features

### 🔢 Sorting Visualizer
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

### 🔍 Searching Visualizer
- Linear Search
- Binary Search

### 🧭 Pathfinding Visualizer
- Dijkstra's Algorithm
- A* (A-Star) Algorithm
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
